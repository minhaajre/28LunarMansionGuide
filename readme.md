# 28 Lunar Mansions Guide

A web interface and guide for exploring the 28 Lunar Mansions. This project maps traditional cosmological frameworks into an accessible, interactive digital format.

## Author
**Minhaaj Rehman**

## Tech Stack
* **Frontend:** Static HTML/CSS/JS
* **Hosting:** Vercel 

## Local Development
To run this project locally, simply open the `index.html` file in any modern web browser. No local server or build process is required.

## Deployment
This project is configured for seamless static deployment on [Vercel](https://vercel.com/). 

The included `vercel.json` handles:
- Enforcing clean URLs 
- Security headers
- Cache invalidation 

To deploy manually via the Vercel CLI:
```bash
npm i -g vercel
vercel
